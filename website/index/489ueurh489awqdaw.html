BAMSEMOMS'S DISCORD COMMAND LIST




Music

!flushqueue Flushes the music queue, removing all songs currently waiting in the queue

!movehere Moves the bot into the same voice channel you're in

!pause Pauses the song that is currently playing

!playlist Main command used to manage the custom music playlists

!queue Lists all the songs in the music queue

!repeatsongs Repeats the songs currently in the music queue

!request [name/url] Request a song via YouTube, SoundCloud, Twitch, radio streams or by name

!resume Resumes the song that was playing before

!shuffle Shuffles the songs waiting in the queue

!skip Skips the song that is currently playing

!volume Changes the volume of the music

!voteskip Votes to skip the song that is currently playing



Utility

!expand Unshorten an URL

!ping Ping, Pong

!rank Displays the users rank, level and XP if Leveling is enabled

!remind Sets a message you would like to be reminded of later

!shorten Shorten a URL using goo.gl



Poll

!yesnopoll Creates a poll people can vote on
